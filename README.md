# CoffeeMoney
いつも決まって買う少額のもの（コーヒー・たばこなど）が、ひと月でいくらになるのかカウントするためのアプリケーション。
すべての支出を管理するのは手間が大きいため、特定の支出の管理だけを手軽に行いたいと考え作成しました。

## １．カテゴリー登録機能
コーヒー・たばこ・お酒・スナック・娯楽費などの支出のカテゴリーを登録する。カテゴリーごとに予算を設定できる。

## ２．アイテム登録機能
カテゴリーごとに具体的なアイテムを登録する。（コンビニコーヒー・スタバコーヒー・コーヒー豆など）アイテムごとに価格を設定できる。

## 3.カウント機能
登録したアイテムボタンをクリックすることで、クリックした日時で支出レコードが作成される。
当月の当該アイテムの支出カウントが＋１される。
カウント画面には、月間予算と残高が表示される。

## 4.一覧表示機能
作成された支出レコードの月別一覧表示ができる。

## 5.レコード削除機能
任意のレコードを削除することができる。

## 6.アイテム・カテゴリーの削除・更新機能
アイテム・カテゴリーの更新・削除が可能。カテゴリー削除時には紐ずくアイテムも同時に削除されるため、削除前に確認のアラートが表示される。

使用技術 *SpringBoot java *postgreSQL
